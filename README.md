## 🚀 Starting

```bash
# Clone this project
$ git clone https://github.com/oleandrosiq/carrossel

# Access
$ cd carrossel

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
